# Sample PDF Documents

Place your PDF documents in this directory for the RAG system to process.

## Supported Files
- Any PDF files (.pdf extension)
- Multiple PDFs can be processed simultaneously
- Documents will be automatically indexed on first run

## Examples
- Research papers
- Technical documentation
- Manuals
- Reports
- Books
- Any text-based PDF content

## Note
After adding PDFs, restart the application to index the new documents.
Or use the `/api/reindex` endpoint to reindex without restart.
